# Client-Side Ethereum Integration (ETH)

This document explains how the **client (frontend)** interacts with Ethereum using wallets like MetaMask and libraries such as `ethers.js`.

---

## 1. Prerequisites

- A modern browser (Chrome, Brave, Firefox)
- Wallet extension (MetaMask recommended)
- Basic JavaScript / TypeScript knowledge
- Node.js installed

---

## 2. Wallet Connection (MetaMask)

### Install MetaMask
- Download from: https://metamask.io
- Create or import a wallet
- Switch to **Ethereum Mainnet** or desired network

---
